## Part channel

```cURL
curl -X POST "https://api.nightbot.tv/1/channel/part" \
  -H "Authorization: Bearer 4fb1fed8889ec9d1c319d5b3c9a54b23"

{
    "status": 200
}
```

Makes Nightbot part (leave) the current user's channel

**HTTP Request**

`GET https://api.nightbot.tv/1/channel/part`

**Scope**

`channel`
